# Поставки — учёт отгрузок

Личный инструмент учёта поставок (маршрут, груз, даты, план/факт по деньгам в разных валютах). Один файл — `index.html`, ничего дополнительно ставить не нужно.

## Как разместить на GitHub Pages

1. Создайте новый репозиторий (**сделайте его Private**, см. пояснение про пароль ниже).
2. Загрузите `index.html` в корень репозитория.
3. Зайдите в **Settings → Pages**, в разделе Source выберите ветку `main` и папку `/root`, сохраните.
4. Через минуту-две сайт будет доступен по адресу вида `https://<ваш-логин>.github.io/<название-репозитория>/`.

## Важно про хранение данных

- Внутри Claude.ai данные сохраняются в привязанном к аккаунту хранилище.
- Вне Claude (на GitHub Pages) инструмент автоматически переключается на `localStorage` — это тоже работает, но:
  - данные привязаны **к конкретному браузеру на конкретном устройстве**, не синхронизируются между компьютером и телефоном;
  - если очистить историю/данные сайтов в браузере — таблица обнулится.
- Поэтому: регулярно нажимайте **«Скачать Excel»** — это ваш настоящий бэкап, который не зависит от браузера.

## Пароль входа

Пароль (`9761`) зашит прямо в код страницы в открытом виде. Это отсекает случайный взгляд, но не защита от того, кто откроет исходный код страницы. Если репозиторий будет публичным — пароль увидит любой. Держите репозиторий **Private**, если это важно.

## Файлы

- `index.html` — вся страница целиком (разметка, стили, логика). Ничего больше грузить не нужно, библиотека для Excel (`SheetJS`) подключается по ссылке из интернета автоматически.
